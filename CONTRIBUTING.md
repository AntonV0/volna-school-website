# Contributing

Status: current as of 2026-05-04.

This project is currently maintained as a private product codebase rather than a general open-source project.

Public contributions are not being accepted at this time. Issues and pull requests may be closed without review if they are outside the current rebuild workflow.

If you are reviewing this repository for hiring, collaboration, or educational purposes, the most useful starting points are:

- `README.md` for project context and architecture.
- `docs/public-repo-checklist.md` for public-repo hygiene.
- `docs/deployment-setup.md` for deployment and environment setup notes.
- `.github/pull_request_template.md` for the expected review checklist.

Do not submit or request private business data, staff/client information, credentials, raw migration assets, or unreviewed media through this repository.
