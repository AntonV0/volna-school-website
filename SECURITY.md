# Security

This repository is a public portfolio view of a real business website rebuild.

Do not open public issues, pull requests, or discussions containing:

- Credentials, tokens, keys, or environment values
- Supabase project details beyond public-safe variable names
- Admin, teacher, student, parent, or trial registration data
- Private business records or unpublished operational details
- Raw migration assets, screenshots, staff images, or student/client media

If you believe you have found a security issue, report it privately to the repository owner or project maintainer. Do not include exploit details, private data, screenshots of private areas, or secret values in public GitHub surfaces.

For local development, keep real configuration only in ignored files such as `.env.local` and in deployment provider environment settings.
